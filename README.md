## ğŸš€ Getting Started

1. **Prerequisites**
   - Node.js (v14 or newer)
   - npm or yarn
   - Expo CLI

2. **Installation**
   ```bash
   # Clone the repository
   git clone https://github.com/yourusername/starvest.git
   cd starvest

   # Install dependencies

   # or

   # Start the development server
   ```

   - Scan QR code with Expo Go app on your device
